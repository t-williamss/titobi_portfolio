Writing

How to add a new piece of writing:

  1. Make a copy of the writing-template.html file.
  2. Rename the copied file to something descriptive (e.g., "my-new-story.html").
  3. Open the new file and change the title and content to your liking.
  4. Add a link to your new file below.
